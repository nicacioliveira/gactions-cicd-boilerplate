### | Running api:

```shell
yarn && yarn dev
```

### | Running tests

Unit tests:

```
yarn test:unit
```

Functional tests:

```
yarn test:functional
```

All tests:

```
yarn test

```

### | Building and running

-- 1

```
yarn build
```

-- 2

```
yarn start
```

or

```
node ./dist/index.ts
```
